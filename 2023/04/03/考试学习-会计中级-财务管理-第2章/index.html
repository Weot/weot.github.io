第二章 财务管理基础

发表于 2023-04-03 更新于 2023-04-22 分类于 考试学习 ， 会计中级 ， 财务管理 阅读次数：
本章属于重点章节, 主要介绍了财务管理的基础知识, 为后面章节中的各种个计算奠定基础. 客观题和计算分析题为主, 预计2023年5-7分.

第二章 财务管理基础

第一节 货币时间价值

01 货币时间价值的概念

短期国债利率, 可以看做是纯粹利率

02 终值和现值的计算

复利终值:$F=P(F/P, i, n)$

复利现值:$P = F(P/F, i, n)$

普通年金终值:$F = A(F/A, i, n)$

年偿债基金:$A = F(A/F, i, n)$年偿债基金系数与普通年金终值系数互为倒数

普通年金现值: $P = A(P/A, i, n)$

年资本回收额:$A = P(A/P, i, n)$ 每月房贷的计算方法

预付年金终值: 普通年金终值*(1+i)

预付年金现值: 普通年金现值*(1+i)

递延年金终值:$F = A(F/A, i, n )$, 与普通年金终值相同

递延年金现值: $P = A(P/A, i, n) * (P/F, i, m)$

永续年金: P = A / i (期末) P = A / i+A (期初)

03 利率的计算

  1. 现值或终值系数已知的利率计算

    插值法:$\frac {i-i_1} {i_2-i_1} = \frac {\alpha-\beta_1} {\beta_2-\beta_1}$

  2. 现值或终值系数未知的利率计算

    逐步测试, 插值法

  3. 实际利率计算

    实际利率i, 名义利率r, 复利次数m

    $i = (1+ \frac{r}{m})^m-1$

  4. 通货膨胀情况下的实际利率

    1 + 名义利率 = (1 + 实际利率) * (1 + 通货膨胀率)

    $实际利率=\frac{名义利率-通货膨胀利率} {1+通货膨胀利率}$

第二节 风险与收益

01 资产收益与收益率

  1. 资产收益含义

    $ 资产收益额 / 期初资产价值 = 资产的收益率 $

    资产收益额 = 利息/红利/股息收益 + 资本利得

    资本利得 = 期末资产价值 - 期初资产价值

  2. 类型

    1. 实际收益率

      已经实现或者确定可以实现的收益率(扣除通货膨胀影响)

    2. 预期收益率

      不确定条件下的

      预期收益率$E(R) = \sum^{n}_{i=1}(P_i \times R_i)$

      $P_i$表示出现概率,$R_i$表示对应收益率

    3. 必要收益率

      也成为最低报酬或最低要求收益率

      必要收益率 = 无风险收益率 + 风险收益率

      无风险收益率 = 纯粹利率 + 通货膨胀补偿利率, 短期国债利率可以近似替代无风险收益率

02 资产的风险及其衡量

  1. 风险的概念

    • 收益不确定性
    • 使企业的实际收益与预期收益发生背离

  2. 风险的衡量

    • 方差:$\delta^2 = \sum ^n _i(X_i - \overline E )^2 \times P_i$,

      • $\overline E$: 期望值

      • $X_i$: i情况下下的收益率

      • $P_i$: 对应情况发生的概率

    • 标准差: 方差平方根

    • 标准差率:$V=\delta / \overline E \times %$, 标准差与期望之比

      标准差/方差: 适合期望相同的决策风险程度比较, 值越大, 风险越大

      标准差率: 期望值相同或不同都可以比较, 值越大, 风险越大

  3. 风险矩阵

    • 横坐标: 发生概率: 极低, 低, 中等, 高,

    • 纵坐标: 影响程度: 可忽略, 微小, 中度, 严重

      优点: 描绘出风险重要性等级, 可视化

      缺点: 主观, 影响使用的准确性

03 风险管理

  1. 风险管理原则
    • 战略性
    • 全员性
    • 专业性
    • 二重性
    • 系统性
  2. 风险管理对策
    • 风险规避
    • 风险承担
    • 风险转移
    • 风险转换
    • 风险对冲
    • 风险补偿
    • 风险控制

04 证券资产组合的收益与风险

  1. 证券资产组合的预期收益率

    各种资产收益率的加权平均数
    $R_p = \sum ^n _i W_i \times R_i$

    $R_p$: 预期收益率

    $W_i$: 权重(价值比例)

    $R_i$: 对应资产的收益率

  2. 证券资产组合的风险及其衡量

    1. 组合分风险分散功能

      两项组合收益率方差$\delta_p^2 = W_1^2\delta_1^2 + W_2^2\delta_2^2 + 2W_1W_2 \rho_{1,2}\delta_1\delta_2$

      简化记忆$A^2+B^2+2AB*\rho_{1, 2}$, 类似平方差或平方和的公式

      $\rho_{1,2}$​ 代表两项资产收益率的相关程度, [-1, 1]

      1: 完全正相关, 不能降低任何风险

      -1: 完全负相关, 风险最小, 最大程度降低风险

      -1~1: 不完全相关关系, 可以分散风险

    2. 非系统性风险和系统性风险

      • 非系统性风险: 发生于个别公司, 可分散风险
      • 系统性风险: 市场风险, 不可分散风险

    3. 系统风险的衡量

      $\beta$​ 系数: 某资产的系统风险是市场组合风险的多少倍

      无风险资产:$\beta$系数为0

      $\beta > 0$: 收益率变化方向与市场平均收益率方向相同

      $\beta > 1$: 收益率变化大于市场平均收益率波动幅度

      $\beta = 1$: 等于市场平均收益率波动幅度

      $0 < \beta < 1$: 小于市场平均收益率波动幅度

      $\beta < 0$: 与市场平均收益率方向相反

      证券组合的$\beta$系数:$\beta_p = \sum_i^n (\beta_i \times W_i)$

05 资本资产的定价模型

资本资产: 主要为股票资产

$某资产的必要收益率R = 无风险收益率R_f + 风险收益率$

$风险收益率 = 该资产系统风险系数 \beta \times (市场组合收益率R_m - 无风险收益率R_f)$

$R = R_f + \beta*(R_m - R_f)$

$R_m$: 市场组合收益率, 平均风险的必要收益率, 市场组合的必要收益率

$R_f$: 纯粹利率 + 通货膨胀补偿率

$(R_m - R_f)$: 市场风险溢酬, 市场组合的风险收益率, 股票市场的风险收益率, 平均风险的风险收益率

第三节 成本性态分析

01 固定成本

  1. 固定成本的含义及特征:
    1. 含义: 一定期间总额保持相对稳定成本
    2. 特征:
      • 一定范围内固定成本总额保持不变
      • 单位固定成本与业务量增减方向呈反向变动
  2. 分类:
    1. 约束性固定成本
      • 车辆交强险
      • 房屋租金
      • 固定资产设备折旧
      • 管理人员的基本工资
    2. 酌量性固定成本
      • 广告费
      • 职工培训费
      • 新产品研究开发费用

02 变动成本

  1. 含义及特征
    1. 含义: 总额随业务量的变动而成正比例变动的成本
    2. 特征
      • 正比例变动
      • 单位变动成本不变
  2. 分类
    1. 技术性变动成本: 经理人员不能决定发生额
      • 直接材料
      • 直接人工
    2. 酌量性变动成本: 决策行动可以改变
      • 百分比支付的销售佣金
      • 新产品研制费用
      • 技术转让费用

03 混合成本

  1. 含义及特征

    1. 含义: 混合固定成本与变动成本
    2. 特征:
      • 随业务量变化而变化
      • 不与业务量保持纯粹的正比例关系

  2. 分类

    1. 半变动成本

      初始基数+变动成本

      电话费等

    2. 半固定成本

      阶梯式变动成本, 一定业务量内发生额固定, 超过一定业务量, 变为另一个固定成本

      企业管理员, 运货员, 检验员的工资

    3. 延期变动成本

      一定范围内固定基数+变动成本

      职工的基本工资+加班工资, 手机流量费用等

    4. 曲线变动成本

      • 递增曲线成本

        累进计件工资, 违约金

      • 递减曲线成本

        有折扣或优惠条件的水, 电消费成本

        费用封顶的通讯服务等

  3. 混合成本的分解方法

    1. 分解公式

      $y = a + b*x$

      $y$: 总成本

      $a$: 固定成本总额

      $b$: 单位变动成本

      $x$: 业务量

    2. 分解方法

      • 高低点法

        $b = \frac {y_高 - y_低} {x_高 - x_低}$

        需要历史数据, 代表性差

      • 线性回归法

        使用最小二乘法, 确定回归直线的参数$a, b$

        需要历史数据

      • 账户分析法

        根据相关成本账户和明细账, 结合其业务量的依存关系

        简便易行, 但比较粗糙且带有主观判断

      • 工业工程法

        工业工程研究方法,

        适用收入成本与产出数量有规律关系的成本分解

      • 合同确认法

        根据合同或协议规定

        需配合账户分析法使用

本章属于非重点章节, 主要介绍筹资管理的基本理论, 涉及股权筹资, 债务筹资, 衍生工具筹及筹资实务创新等内容, 重点为筹资的分类, 各种个不同筹资方式的特点, 租金的计算等. 2023年预计7分左右.

第四章 筹资管理(上)

第一节 筹资管理概述

01 企业筹资的动机

  1. 创立性筹资动机
  2. 支付性筹资动机
  3. 扩张性筹资动机
  4. 调整性筹资动机
  5. 混合型筹资动机

02 筹资管理的内容

  1. 科学的预计资金的需求量
  2. 合理安排筹资的渠道, 选择筹资方式
    • 筹资渠道
      • 国家财政投入和政府补贴
      • 银行与非金融机构信贷
      • 资本市场筹集
      • 其他法人单位与自然人的投入
      • 企业自身积累
    • 筹资方式
      • 股权筹资
      • 债务筹资
  3. 降低资本成本, 控制财务风险
    • 资本成本
      • 筹资费用
        • 股票发行费用
        • 借款手续费
        • 证券印刷费
        • 公证费
        • 律师费
      • 占用费用
        • 利息支出
        • 股利支出
        • 租赁的资金利息等费用
    • 财务风险
      • 偿债风险

03 筹资方式

  1. 股权筹资
    • 吸收直接投资
    • 发行股票
    • 留存收益
  2. 债务筹资
    • 商业信用
    • 租赁
    • 金融机构借款
    • 发行债券
  3. 混合筹资
    • 发行可转换债券
    • 发行优先股股票

04 筹资的分类

  1. 按资金权益性质
    • 股权筹资
    • 债务筹资
    • 衍生工具筹资
  2. 是否以金融机构为媒介
    • 直接筹资
    • 间接筹资
  3. 按资金来源范围不同
    • 内部筹资
    • 外部筹资
  4. 按适用期限不同
    • 长期筹资
    • 短期筹资

05 筹资管理的原则

  1. 筹资合法
  2. 规模得当
  3. 取得及时
  4. 来源经济
  5. 结构合理

第二节 债务筹资

01 银行借款

  1. 银行借款的种类

    • 按提供贷款的机构

      • 政策性银行贷款
      • 商业银行贷款
      • 其他金融机构贷款

    • 按有无担保要求

      • 信用贷款
      • 担保贷款
        • 保证贷款: 第三人担保
        • 抵押贷款: 不转移财产占有, 不动产, 动产
        • 质押贷款: 转移财产占有, 动产或财产权利, 汇票, 股票, 支票, 债券, 存款单, 提单, 可转让股份, 商标专用权, 专利权, 著作权

  2. 长期借款的保护性条款

    • 例行性保护条款

      • 定期提交财务报表
      • 及时偿债
      • 保持存货储备量
      • 不准以资产作其他承诺的担保或抵押
      • 不准贴现应收票据或出售应收账款

    • 一般性保护条款
      对资产流动性及偿债能力等方面的要求

      • 保持: 资产流动性
      • 限制
        • 非经营性支出
        • 资本支出的规模
        • 长期投资
        • 再举债规模
      • 特殊性保护条款
        • 主要领导购买人身保险
        • 借款用途不得改变
        • 违约惩罚性条款

  3. 银行借款的筹资特点

    优点:

    • 筹资速度快

    • 资本成本较低

    • 筹资弹性较大

    缺点:

    • 限制条款多

    • 筹资数额有限

02 发行公司债券

  1. 公司债券的含义

    依照法定程序发行, 约定在一定期限内还本付息的有价证券

    公开发行, 非公开发行

    公开发行所募集资金不得用于弥补亏损和非生产性支出

  2. 公司债券的种类

    • 按是否匿名
      • 记名债券
      • 无记名债券
    • 是否转换为公司股权
      • 可转换债券
      • 不可转换债券
    • 按有无特定资产担保
      • 担保债券
        • 不动产抵押债券
        • 动产抵押债券
        • 证券信托抵押债券
      • 信用债券
    • 按是否公开发行
      • 公开发行债券
      • 非公开发行债券

  3. 公司债券的偿还及特点

    • 偿还
      • 提前偿还: 资金结余, 预测利率下行
      • 到期分批偿还
      • 到期一次偿还
    • 特点(与银行借款相比)
      • 一次筹资数额大
      • 筹资使用限制少
      • 资本成本较高
      • 提高公司社会声誉

03 租赁

  1. 租赁的分类
    • 短期租赁和低价值资产租赁: 直线法
    • 其他租赁: 确认使用权资产和租赁负债
      • 使用权资产
      • 租赁负债
  2. 租赁的基本特征
    • 所有权和使用权分离
    • 融资与融物相结合
    • 租金的分期支付
  3. 租赁的基本形式
    • 直接租赁
    • 售后回租
    • 杠杆租赁
  4. 租赁租金的计算
    • 决定因素
      • 设备原价及预计残值
      • 利息
      • 租赁手续费和利润
    • 租赁的计算(年金现值)
      • 折现率 = 利率 + 租赁手续费率
      • 残值归租赁公司: 设备价款 = 租金的现值 + 残值的现值
      • 残值归承租公司: 设备价款 = 租金的现值
  5. 租赁的筹资特点
    • 无需大量资金就能迅速获得资产
    • 财务风险小, 财务优势明显
    • 筹资的限制条件少
    • 能延长资金融通的期限
    • 资本成本较高

04 债务筹资的优缺点

  1. 优点
    • 筹资速度快
    • 筹资弹性较大
    • 资本成本较低
    • 可以利用财务杠杆
    • 稳定公司的控制权
  2. 缺点
    • 不能形成企业稳定的资本基础
    • 财务风险较大
    • 筹资数额有限

资本成本大小: 银行借款<公司债券<租赁

第三节 股权筹资

01 吸收直接投资

  1. 出资方式
    • 货币资产
    • 非货币资产
      • 知识产权
        • 出资风险较大
        • 不得以劳务, 信用, 自然人姓名, 商誉, 特许经营权, 设定担保的财产等作价出资
      • 特定债权
        • 上市公司可转换债券
        • 国有或国有控股企业债权
        • 企业公司制改制, 经银行外其他债权人协商同意, 将债权转为股权
        • 国企境内债权人转让债权给境外投资机构, 债权转股权
        • 国企改制, 所欠职工工资, 职工集资款, 职工自愿条件下, 转为一定股权
  2. 特点
    • 能够尽快形成生产力
    • 容易进行信息沟通
    • 资本成本较高
    • 公司控制权集中, 不利于公司治理
    • 不易进行产权交易

02 发行普通股票

  1. 股票的含义, 特征与种类

    • 股票的含义: 所有权凭证

    • 股票的特征

      • 永久性
      • 流通性
      • 风险性
      • 参与性

    • 股东权利

      • 公司管理权
      • 收益分享权
      • 股份转让权
      • 优先认股权
      • 剩余财产要求权

    • 股票的种类

      • 按股东权利
        • 普通股票
        • 优先股股票: 股利优先分配权, 剩余资产优先权, 参与公司管理有一定限制 (两优先一控制)
      • 按票面有无记名
      • 按发行对象和上市地点 A股, B股, H股, N股, S股

  2. 我国证券交易所概况

    • 上交所
      • 主板
      • 科创板
    • 深交所
      • 主板
      • 中小企业版
      • 创业板
    • 北交所

  3. 首次公开发行股票的条件(IPO)

    • 各版块IPO条件
      • 主板: 核准制
        • 3年正数净利润, 且累计超过3000万
        • 3年现金流净额累计超过5000万或3年累计营收3亿
        • 发行股本总额不少于3000万
        • 无形资产占净资产不高于20%
        • 最近一期不存在未弥补亏损
      • 创业板: 试行注册制
        • 具备健全且运行良好的组织机构
        • 具有持续具有能力
        • 3年财务报告由注册会计师出具无保留意见的审计报告
        • 监管机构规定的其他条件
      • 科创板: 试行注册制, 同创业板

  4. 上市公司股票发行条件

    • 公开发行
      • 组织机构健全, 运行良好
      • 盈利能力具有可持续性
      • 财务状况良好
      • 最近36个月内财务会计文件无虚假记载, 且不存在重大违法行为
      • 不存在在其他不得公开发行股票的其他行为
    • 非公开发行
      • 发行价格不低于基准日前20个交易日公司股票均价的80%
      • 6个月内不得转让, 控股东, 实际控制人及其控制的企业认购的股份, 18个月内不得转让
      • 募集资金的使用符合相关规定
      • 导致公司控制权发生变化的, 还应该符合证监会的其他规定
      • 发行对象不超过35名

  5. 新三板公开发行股票的条件

    • 新三板的含义: 全国中小企业股份转让系统
    • 向不特定合格投资者公开发行股票的发行人所需求的条件
      • 12月的创新层挂牌公司
      • <公司公众法> <分层管理办法>
      • 发行对象应为开通交易权限的合格投资者

  6. 股票的发行方式

    • 间接发行

      通过中介机构间接向社会公众公开发行

      • 优点: 发行范围广, 发行对象多, 易于足额筹集资本, 有利于提高公司知名度
      • 缺点: 审批手续复杂严格, 发行成本高

    • 非公开直接发行

       只向少数特定对象直接发行股票

  7. 上市公司的股票发行

    • 配股

      向原始股东配售股票

    • 增发

      • 公开增发: 向社会公众发售
      • 非公开增发(定向增发)
        • 有利于引入战略投资者和机构投资者
        • 有利于市值估值溢价, 提升母公司的资产价值
        • 并购手段, 有利于集团整体上市, 减轻并购的现金流压力

  8. 股票发行中引入战略投资者的意义

    • 战略投资者的概念和要求

      • 要与公司经营业务联系紧密
      • 要出于长期投资目的而较长时间地持有股票
      • 要有相当的资金实力, 且持有股票数量较多

    • 引入战略投资者的意义

      • 提升公司形象, 提高资本市场认同度
      • 优化股权结构, 健全公司法人治理
      • 提高公司资源整合能力, 增强公司的核心竞争力
      • 达到阶段性的融资目标, 加快上市融资进程

  9. 股票的上市交易与退市

    • 股票上市的目的
      • 便于筹措新资金
      • 促进股权流通和转让
      • 确定公司价值
    • 股票上市交易的缺点
      • 上市成本高
      • 公司将负担较大的信息披露成本
      • 可能暴露商业机密
      • 股价有时会歪曲公司的实际情况
      • 可能会分散公司控制权, 造成管理上困难
    • 股票上市条件
    • 退市的种类
      • 强制退市: 交易类强制退市, 财务类, 规范类, 重大违法类
      • 主动退市

  10. 发行普通股股票的筹资特点

    • 优点
      • 两权分离
      • 增强公司的社会信誉, 促进股权的流通和转让
    • 缺点
      • 资本成本较高
      • 不易及时形成生产能力

03 留存收益

  1. 性质: 所有者权益
  2. 筹资途径
    • 提取盈余公积金
    • 未分配利润
  3. 特点
    • 不用发生筹资费用
    • 维持公司的控制权分布
    • 筹资数额有限

三种股权筹资方式资本成本:

  • 筹资费: 普通股 > 吸收直接投资 > 留存收益
  • 占用费: 吸收直接投资 > 普通股 = 留存收益
  • 资本成本: 吸收直接投资 > 普通股 > 留存收益

04 股权筹资的优缺点

  1. 优点
    • 企业稳定的资本基础
    • 企业良好信誉的基础
    • 财务风险小
  2. 缺点
    • 资本成本较高
    • 控制权变更影响
    • 信息沟通与披露成本较高

第四节 衍生工具筹资

01 可转换债券

  1. 含义: 普通债券与证券期权的结合体
  2. 分类
    • 不可分离的
    • 可分离的
  3. 基本性质
    • 证券的期权性: 未来的买入期权
    • 资本转换性
    • 赎回与回售
      • 赎回: 股价连续高于转股价格达到某一幅度
      • 回售: 股价连续低于转股价格达到某一幅度
  4. 基本要素
    • 标的股票
    • 票面利率
    • 转换价格
    • 转换比率
    • 转换期: 发行后6个月
    • 赎回条款: 加速条款, 保护发债发行公司
    • 回售条款: 卖权, 减低持券人的持券风险
    • 强制转换条款
  5. 筹资特点
    • 优点:
      • 筹资灵活
      • 资本成本较低: 部分利率低于银行贷款利率, 资本成本高于债务筹资, 普遍低于股权筹资
      • 筹资效率高
    • 缺点:
      • 存在一定财务压力: 到期不转换或者回售

02 认证股权

  1. 含义: 约定价格认购
  2. 基本性质
    • 期权性
    • 内在价值的投资工具
  3. 筹资特点
    • 融资促进工具
    • 改善上市公司的治理结构
    • 股权激励机制

03 优先股

  1. 含义: 优于普通股

  2. 基本性质

    • 约定股息
    • 权利优先
      • 优先分配股息
      • 优先分配剩余财产
    • 权利范围小: 无表决权(与优先股股东有关的, 有限表决权)

  3. 种类

    我国的规定

    • 固定股息 与 浮动股息
    • 强制分红 与 非强制分红
    • 累积优先股 与 非累积优先股
    • 非参与优先股 与 参与优先股
    • 不可转换 与 可转换
    • 票面金额100元

  4. 特点

    • 有利于丰富资本市场的投资结构
    • 有利于股份公司股权结构的调整
    • 有利于保证普通股收益和控制权
    • 有利于降低公司财务风险
    • 可能给公司带来一定的财务压力

第五节 筹资实务创新

  1. 非公开定向债务融资工具
  2. 私募股权投资
  3. 产业基金
  4. 商业票据融资

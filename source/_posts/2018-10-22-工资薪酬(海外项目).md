## 前言

海外职工薪酬包含企业自身的职工(合同化, 市场化), 劳务工, 协作工, 人才派遣, 海外雇工等多种形式, 因此, 稍许复杂. 

基本做法, 将企业自身的员工过应付职工薪酬, 计入工程施工-人工费, 国内服务部职工薪酬计入制造费用-人工费, 其他形式的用工直接计入工程施工-分包费.

## 一. 国内人员薪酬

### 1. 国内人员工资发放

```python
借: 应付职工薪酬-工资
    制造费用-差旅费    # 误餐补助
    应付职工薪酬-劳务费    # 协作, 派遣, 劳务工
    
    贷: 其他应付款-职工保险
                 -职工未领款
```
工资正式发放时, 核销相应的`其他应付款-职工未领款`, 未核销掉的为挂账的工资

```python
借: 其他应付款-职工未领款
    贷: 应付内部单位款-往来款
```

### 2. 国内人员工资计提

```python
借: 制造费用-人工费
    制造费用-劳务费    # 协作, 派遣, 劳务工
    
    贷: 应付职工薪酬
        内部交易过渡科目-工资    # 协作, 派遣, 劳务工
```

## 二. 国外人员薪酬

### 1. 国外人员工资发放

核算时都进入`其他应付款-职工未领款`, CPECC代发后转交易, 进行核销

转交易后用外币过渡科目过渡
```python
借: 外币过渡科目(迪拉姆)
    贷: 应付内部单位款-往来款
```

核销相应的`其他应收款-职工未领款`
```python
借: 其他应付款-职工未领款
    库存现金    #  从CP领取现金支票发放工资
    贷: 外币过渡科目(迪拉姆)
    
借: 其他应付款-职工未领款
    贷: 库存现金
    
# 扣除罚款等费用
借: 其他应付款-职工未领款
    贷: 其他应收款-应向职工收取垫付款项
```

#### 1) 项目部部分

```python
借: 应付职工薪酬-工资
    内部交易过渡科目-工资    # 协作, 派遣, 劳务工
    
    贷: 其他应付款-职工保险
        外币过渡科目(人民币)


借: 外币过渡科目(迪拉姆)

    贷: 其他应付款-职工未领款(迪拉姆)
    

借: 应付职工薪酬-奖金/补贴
    内部交易过渡科目-工资
    
    贷: 其他应付款-职工未领款(迪拉姆)
    
```

#### 2) 研发部分

```python
借: 应付职工薪酬-工资
    
    贷: 其他应付款-职工保险
        外币过渡科目(人民币)

借: 外币过渡科目(迪拉姆)

    贷: 其他应付款-职工未领款(迪拉姆)
    
借: 应付职工薪酬-奖金/补贴
    
    贷: 其他应付款-职工未领款(迪拉姆)

```

#### 3) 外籍员工

```python
借: 内部交易过渡科目-其他

    贷: 其他应付款-职工未领款
```

#### 4) 人力公司部分

```
借: 预付账款-劳务款
    
    贷: 其他应付款-职工保险
        外币过渡科目(人民币)
    
借: 外币过渡科目(迪拉姆)
    
    贷: 其他应付款-职工未领款(迪拉姆)
```

保险及管理费部分, 分公司代付后转交易

```python
借: 应付账款-劳务款
    其他应付款-职工保险    # 核销发放时挂出的保险
    
    贷: 预付账款-劳务款
        应付内部单位款-往来款
```

### 2. 国外人员工资计提

#### 1) 项目部部分

```python
借: 工程施工-人工费
    
    贷: 应付职工薪酬(提取或明细)
    
# 协作, 派遣, 劳务工    
借: 工程施工-分包费    
    
    贷: 内部交易过渡科目-工资
```

#### 2) 研发部分

```python
借: 研发支出-人工费
    
    贷: 应付职工薪酬
```

#### 3) 外籍员工
```python
借: 工程施工-分包费
    辅助生产成本-分包费
    长期待摊费用-临设
    
    贷: 内部交易过渡科目-其他
```

## 三. 各项保险及公积金

### 1. 个人承担部分

个人承担部分, 在公司发放时候挂`其他应付款-职工保险`.
1. 合同化, 市场化, 劳务工, 企业化由分公司代付(国内为项目自己付款)后转交易
2. 协作工个人保险在公司转企业保险时一并转来

```python
借: 其他应付款-职工保险
    贷: 应付内部单位款-往来款
```

### 2. 企业承担部分

企业承担部分, 公司每月会转过来, 各项保险及公积金是按上月工资一定比例提取的

分公司转保险交易
```python
借: 内部交易过渡科目-工资
    贷: 应付内部单位款-往来款
```

核算各项保险
```python
借: 应付职工薪酬-各项保险明细
    工程施工-分包费
    其他应付款-协作工保险    # 核销协作工保险
    贷: 内部交易过渡科目-工资
```

计提各项保险
```python
借: 工程施工-人工费
    贷: 应付职工薪酬-各项保险明细
```

其他单位核对协作工保险后, 分公司会转交易
```python
借: 其他应付款-协作工保险
    贷: 应付内部单位款-往来款
```

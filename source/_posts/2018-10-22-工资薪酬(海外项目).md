## 前言

工资薪酬的核算总体来说不算复杂, 但是海外项目的用工形式比较多, 且成本区分比较详细, 因此稍许繁琐.
<!--more-->
关于应付职工薪酬用工类型的选择如下:

| 员工种类 | 用工形式选择 |
|---|---|
| 合同化 | 境外-中方-合同化 |
| 市场化 | 境外-中方-市场化 |
| 劳务工 | 境外-中方-劳务派遣工 |
| 集体工 | 境外-中方-劳务派遣工 |
| 协作工 | 境外-中方-劳务派遣工 |
| 人才派遣 | 境外-中方-劳务派遣工 |
| 外籍员工 | 境外-外放-非全日制用工 |

关于成本的分配, 不同类型的员工, 也不尽相同, 大致思路是管理层先过制造费用, 作业层直接计入相应工程成本.

| 分类 | 薪酬科目 | 成本科目 |
|------|---------|---------|
|国内服务部员工|应付职工薪酬-应付工资<br>应付职工薪酬-劳务费|制造费用-人工费<br>制造费用-劳务费|
|项目部员工|应付职工薪酬-应付工资<br>应付职工薪酬-劳务费|工程施工-人工费<br>工程施工-劳务费<br>制造费用-人工费<br>制造费用-劳务费|
|外籍员工|应付职工薪酬-劳务费|工程施工-劳务费|

由于工资是CPECC代发, 工资薪酬处理时会暂挂`其他应付款-职工未领款`, 待CPECC转账后核销, 月底账上剩余数据应该与劳资挂账工资金额相同.
具体账务处理会在正文中一一列举, 按实际凭证分段描述.

## 一. 国内服务部员工薪酬

### 1. 国内人员工资发放

国内服务部员工工资由于处理比较简单, 因此, 发放与计提直接在一张凭证上完成.

### 2. 国内人员工资计提

```beancount
借: 应付职工薪酬-工资
    制造费用-差旅费    # 误餐补助
    应付职工薪酬-劳务费    # 协作, 派遣, 劳务工
    
    贷: 其他应付款-职工保险
                 -职工未领款
借: 制造费用-人工费
    制造费用-劳务费    # 协作, 派遣, 劳务工
    
    贷: 应付职工薪酬-工资
        应付职工薪酬-劳务费
```

### 3. 工资正式发放

总部代付工资后由分公司转交易后, 核销相应的`其他应付款-职工未领款`

```beancount
借: 其他应付款-职工未领款
    贷: 应付内部单位款-往来款
```

## 二. 国外人员薪酬

### 1. 项目员工部分

#### a. 工资发放
```python
借: 应付职工薪酬-工资
    应付职工薪酬-劳务费-工资    # 协作, 派遣, 劳务工
    贷: 其他应付款-职工保险
        外币过渡科目(人民币)

借: 外币过渡科目(迪拉姆)
    贷: 其他应付款-职工未领款(迪拉姆)
    
借: 应付职工薪酬-奖金/补贴
    应付职工薪酬-劳务费-奖金/补贴
    贷: 其他应付款-职工未领款(迪拉姆)
```
#### b. 工资计提
```python
借: 制造费用-人工费/劳务费
    工程施工-人工费/劳务费
    贷: 应付职工薪酬(提取或明细)
```

### 2. 研发部分

#### a. 工资发放

```python
借: 应付职工薪酬-工资
    应付职工薪酬-劳务费
    贷: 其他应付款-职工保险
        外币过渡科目(人民币)

借: 外币过渡科目(迪拉姆)
    贷: 其他应付款-职工未领款(迪拉姆)
    
借: 应付职工薪酬-奖金/补贴
    应付职工薪酬-劳务费-奖金/补贴
    贷: 其他应付款-职工未领款(迪拉姆)

借: 研发支出-人工费
    贷: 应付职工薪酬
```

#### b. 工资计提

研发支出直接在各个研发项目`工资发放`同一凭证上进行计提

### 3. 外籍员工

#### a. 工资发放
```python
借: 应付职工薪酬-劳务费
    贷: 其他应付款-职工未领款

借: 制造费用-劳务费 # 阿布扎比项目部外籍
    工程施工-劳务费
    贷: 应付职工薪酬-劳务费
```

#### b. 工资计提

外籍员工直接在工资发放同一凭证上进行计提


### 4. 附加部分(国内分包与劳务分包部分)

由于国内分包与人力公司雇员持CPECC签证, 因此此类员工也应每月发工资, 分包费与劳务费也就按工资的形式进行付款.

####  1). 人力公司部分

```beancount
借: 预付账款-劳务款
    贷: 其他应付款-职工保险
        外币过渡科目(人民币)
    
借: 外币过渡科目(迪拉姆)
    贷: 其他应付款-职工未领款(迪拉姆)

借: 预付账款-劳务款    # 嘉奖部分
    贷: 其他应付款-职工未领款
```

保险及管理费部分, 分公司代付后转交易

```python
借: 应付账款-劳务款
    其他应付款-职工保险    # 核销发放时挂出的保险
    
    贷: 预付账款-劳务款
        应付内部单位款-往来款
```

#### 2). 分包部分

```
借: 预付账款-劳务款
    贷: 外币过渡科目(人民币)

借: 外币过渡科目(迪拉姆)
    贷: 其他应付款-职工未领款
```

### 5. 实际工资发放

需要CPECC代发的工资都在核算时挂`其他应付款-职工未领款`, CPECC代发后转交易, 进行核销;
同时, 需要扣除的罚款也会在此时核销

转交易后用外币过渡科目过渡
```python
借: 内部交易过渡科目
    贷: 应付内部单位款-往来款
```

核销相应的`其他应收款-职工未领款`
```python
借: 库存现金    #  从CP领取现金支票发放工资   
    贷: 内部交易过渡科目

借: 其他应付款-职工未领款
    贷: 库存现金
    
借: 其他应付款-职工未领款   # WPS部分
    贷: 内部交易过渡科目

借: 其他应付款-职工未领款   # UAE Exchange部分
    贷: 内部交易过渡科目

借: 其他应付款-职工未领款   # 扣除罚款等费用
    贷: 其他应收款-应向职工收取垫付款项
```

## 四. 各项保险及公积金

### 1. 个人承担部分

个人承担部分, 在公司发放时候挂`其他应付款-职工保险`.
1. 合同化, 市场化, 劳务工, 企业化由分公司代付(国内为项目自己付款)后转交易
2. 项目部协作工个人保险在公司转企业保险时一并转来
3. 工程处等协作工个人保险会分别转来

```python
借: 其他应付款-职工保险
    贷: 应付内部单位款-往来款
```

其他单位核对协作工保险后, 分公司会转交易
```python
借: 其他应付款-协作工保险
    贷: 应付内部单位款-往来款
```

### 2. 企业承担部分

企业承担部分, 公司每月会转过来, 各项保险及公积金是按上月工资一定比例提取的

分公司转保险交易
```python
借: 内部交易过渡科目-工资
    贷: 应付内部单位款-往来款
```

核算各项保险
```python
借: 应付职工薪酬-各项保险明细
    工程施工-分包费
    其他应付款-协作工保险    # 核销协作工保险
    贷: 内部交易过渡科目-工资
```

计提各项保险
```python
借: 工程施工-人工费
    贷: 应付职工薪酬-各项保险明细
```
